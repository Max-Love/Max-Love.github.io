◆ getInput()

std::string Engine::getInput ( )

Gets the input from the user as a string. The string could be either QUIT which means that the user wants to quit the program. Or it could be comprised of two substrings separated by an underscore. The first substring can be UP or DOWN, denoting if the key is pressed down or if it is released. The second substring denotes the key which was pressed or released.

Returns
The input from the user as a string.
