Detailed Description

Copied from the provided code in Lab-6 This class serves as an interface to the main graphics renderer for our engine.

Serves as a layer of abstraction in between the graphics renderer and our game engine. (Not really though - the GetWind and GetRenderer functions are still coupled to SDL)
